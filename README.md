# IronCoach v2.2 — Núcleo del Atleta

Esta actualización se instala sustituyendo los archivos actuales del repositorio.

## Novedades
- Nueva sección `Núcleo`.
- Identidad y contexto del atleta.
- Disponibilidad semanal.
- Material y recursos.
- Lesiones, limitaciones y preferencias.
- Fortalezas y debilidades percibidas.
- Autoevaluación de seis capacidades:
  - Fuerza
  - Movilidad
  - Core y estabilidad
  - Resistencia
  - Potencia / intensidad
  - Técnica y coordinación
- Mapa híbrido de capacidades:
  - 35% autoevaluación.
  - 65% evidencias del historial reciente.
- Identificación de fortaleza y mayor margen de mejora.
- Comparación del volumen realizado con las horas disponibles.
- El núcleo se incluye en exportación e importación JSON.

## Actualización
1. Haz una copia JSON desde la versión actual.
2. Sustituye todos los archivos de GitHub por los de este ZIP.
3. Confirma el commit.
4. Abre la URL en Safari y recarga.
5. Cierra y vuelve a abrir la PWA.

Los datos se conservan porque no cambian el dominio ni las claves existentes.
