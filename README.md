## Actualizar GitHub
Sustituye en la raíz del repositorio todos los archivos por los incluidos en este ZIP y confirma los cambios.

## Importante en iPhone
La PWA puede conservar temporalmente la versión anterior por caché.
Después de subir la actualización:
1. Abre la URL en Safari.
2. Recarga la página.
3. Cierra IronCoach desde el selector de aplicaciones.
4. Vuelve a abrirla desde el icono.

Los datos actuales permanecen en LocalStorage porque se conserva el mismo dominio y las mismas claves.
